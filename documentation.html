HTML Visualizer — Documentation

← Back to App

This app converts raw HTML into an interactive DOM tree. You can visualize, search, edit nodes, generate clean HTML, and preview output live.

Default theme is Night Mode. Use the top-right toggle in the app to switch Light/Night.

Feature List View

Quick list of everything available in the app:

  1. Parse HTML and render DOM graph
  2. Pan / Zoom / Fit canvas controls
  3. Search nodes by tag, text, and attributes
  4. Right-click context menu actions
  5. Hover/Tap node action card (Add, Edit, Collapse, Delete)
  6. Smart edit modal for tag/attributes/text
  7. Generate, copy, and download final HTML
  8. Live iframe preview with expand/refresh
  9. Sidebar popup drawer from navbar menu
  10. Theme toggle (Night/Light) with saved preference
  11. Floating page scroll helper button
  12. Responsive desktop/tablet/mobile support

How editing works

Pick any node, open the editor, change its tag, attributes, and text (when valid), then save. Tree, stats, generated HTML, and preview stay in sync.

How node operations work

On desktop, use right-click menu. On mobile/touch, tap node to use action card buttons. Both paths support add, edit, collapse/expand, and delete.

How generation and preview work

After any change, click Generate to produce clean HTML. Copy or download it, and open Preview to render output directly inside the app.

Core Features

  • Parse pasted HTML into a tree graph
  • Animated canvas drawing for nodes and edges
  • Pan, zoom, and fit controls
  • Live node stats: Nodes, Edges, Depth, Leaves

Node Operations

  • Right-click node menu: Edit, Add, Collapse, Delete
  • Hover/tap node action card with 4 quick icons
  • Double-click node to open smart editor
  • Collapse/expand subtree support

Editor & Generation

  • Smart tag selector with grouped HTML tags
  • Attribute key/value editor
  • Text content editor (when valid for tag)
  • Generate, copy, and download updated HTML
  • Live iframe preview with refresh/expand/close

UI & Mobile Support

  • Popup sidebar drawer from navbar menu button
  • Canvas stays stable while sidebar opens
  • Bottom-right floating up/down scroll helper
  • Responsive layout for desktop/tablet/mobile
  • Theme toggle (Night/Light) with saved preference

Keyboard Shortcuts

  • Ctrl + F → Focus search
  • Delete / Backspace → Delete selected node (not root)
  • Escape → Close overlay/popup UI (sidebar, menus)

Example Input HTML

Paste this into the sidebar input and click Visualize:

<html>
  <head>
    <title>Demo</title>
  </head>
  <body>
    <header>
      <h1>My Page</h1>
    </header>
    <main>
      <p class="lead">Hello Visualizer</p>
      <ul>
        <li>Item A</li>
        <li>Item B</li>
      </ul>
    </main>
  </body>
</html>
